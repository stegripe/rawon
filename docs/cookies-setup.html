ü§î Why do I need this?

If you're hosting Rawon on cloud providers like OVHcloud, AWS, GCP, Azure, Railway, or other hosting services, you might encounter the error:

"Sign in to confirm you're not a bot"

This happens because the video platform blocks requests from data center IP addresses. By using cookies from a logged-in account, you can bypass this restriction. Don't worry - it's easy to set up!

  • A secondary/throwaway video platform account (NEVER use your main account!)

üìñ How to Export Cookies

Step 1: Create a Throwaway Account

  1. Go to the video platform's account signup page

  2. Create a NEW account specifically for this bot

  3. ‚ö†Ô∏è IMPORTANT: NEVER use your personal/main account!

Step 2: Log in to the Video Platform

  1. Open your browser

  2. Go to the video platform website

  3. Sign in with your throwaway account

  4. Accept any terms if prompted

Step 3: Install Cookies Export Extension

  • For Chrome/Edge: Install Get cookies.txt LOCALLY (recommended) from Chrome Web Store

  • For Firefox: Install cookies.txt from Firefox Add-ons

Step 4: Export Cookies

  1. Make sure you're on the video platform website

  2. Click the cookies extension icon in your browser toolbar

  3. Click Export or Export cookies for this site

  4. Save the file as cookies.txt

Step 5: Add to Rawon

  1. Go to any channel where Rawon can see your messages

  2. Type: !cookies add 1

  3. Attach the cookies.txt file to your message and send

  4. Rawon will confirm the cookie was added!

üîß Troubleshooting

Still getting "Sign in to confirm you're not a bot" errors?

  • Use !cookies list to check cookie status

  • If a cookie shows Failed, try !cookies reset to retry

  • Add more cookies from different accounts for redundancy

All cookies failed?

  • Create new throwaway accounts

  • Export fresh cookies

  • Add them with !cookies add <number>

Account got suspended?

  • This can happen with heavy usage

  • Simply create a new throwaway account

  • Export new cookies and add them
