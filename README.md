# Rawon Pages

> **Website for Rawon Discord Music Bot. Built with [Next.js](https://nextjs.org/), [Tailwind CSS](https://tailwindcss.com/), and [Material UI](https://mui.com/). PWA supported.**

## 🌐 Live Site

Visit [rawon.stegripe.org](https://rawon.stegripe.org) for documentation, guides, and tools.

## 📚 Features

- Multi-language support (English & Bahasa Indonesia)
- Comprehensive documentation (Getting Started, Configuration, Cookies Setup)
- Permission Calculator tool
- PWA support for offline access
- Mobile-friendly responsive design

## 🚀 Development

```bash
# Install dependencies
pnpm install

# Run development server
pnpm dev

# Build for production
pnpm build

# Run linter
pnpm lint
```
